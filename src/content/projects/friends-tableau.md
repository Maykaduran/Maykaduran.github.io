## Project Overview

**Friends Insights** is a data visualization project focused on analyzing the popular TV show **Friends** using **Tableau**.  
The goal of the project is to transform raw episode and dialogue data into clear and engaging visual insights.

Through interactive dashboards, the project explores patterns in character dialogue distribution, episode structure, and character prominence throughout the seasons.

This project was developed as part of my **dual training program**, applying **data analysis and business intelligence techniques** to entertainment data.

## Dataset Description

The dataset contains structured information about the TV show **Friends**, including:

- Episode titles and seasons
- Characters appearing in each episode
- Dialogue counts per character
- Episode distribution across seasons

This data enables analysis of **character prominence**, **dialogue patterns**, and **narrative dynamics** throughout the series.

---

## Dashboard Insights

The Tableau dashboard provides several visual perspectives on the dataset.

### Character Dialogue Distribution

- Comparison of dialogue volume between the six main characters
- Identification of characters with the highest number of spoken lines

### Season Analysis

- Evolution of character presence across seasons
- Patterns in episode production and character focus

### Episode-Level Analysis

- Episodes with the highest dialogue concentration
- Distribution of interactions between characters

These visualizations allow users to explore the data interactively and uncover patterns in the storytelling structure of the series.
