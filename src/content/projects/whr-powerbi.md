## Project Overview

This project explores the **World Happiness Report** dataset to analyse the factors that influence wellbeing and happiness across countries.

The objective was to apply a full **EDA → ETL → Visualization workflow**, combining Python analysis with BI tools such as Tableau and Power BI to communicate insights clearly.

The project was developed during the **Data Analytics & AI Bootcamp at Adalab**.

## Objectives

- Explore and understand the World Happiness Report dataset
- Clean and transform data using Python
- Create clear visualizations and dashboards
- Communicate relevant insights through Business Intelligence tools
- Build a flexible base for adding future datasets

---

## Methodology

The project follows a structured data workflow:

**EDA & ETL**

- Data exploration using Python
- Cleaning and transformation with Pandas
- Data validation and preparation for BI tools

**Visualization**

- **Tableau** for exploratory visual analysis
- **Power BI** for interactive dashboards and storytelling

**Code Organization**

The repository includes reusable Python scripts organized inside a `src` folder.

---

- **Dataset**: https://www.kaggle.com/datasets/hopesb/whr-from-2005-2022

---

- **KeyInsights**

    - Nordic countries consistently rank among the happiest globally
    - GDP and social support show strong correlation with happiness score
    - Regional differences highlight the role of social and economic stability
