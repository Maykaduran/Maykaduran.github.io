This project includes two interactive console games built in Python as part of the **Adalab Data Analytics Bootcamp**.

The goal was to practice programming fundamentals such as control flow, data structures, and randomness while building simple but functional applications.

## Objectives

Build a Python-based Rock, Paper, Scissors game that allows a user to play against a bot and Capitalisimo is a develop a Python quiz game where the program asks the user questions about world capitals.

---

## Projects Included

### Rock, Paper, Scissors

A console-based implementation of the classic game where the player competes against a computer-controlled bot.

**Main features**

- Best-of-three rounds gameplay
- Randomized bot decisions
- Score tracking system
- Game loop with victory conditions

### Capitalísimo

A geography quiz game designed to test knowledge of world capitals.

**Main features**

- Random country selection
- Score system with penalties
- Dictionary-based data structure for country–capital mapping
- Prevention of repeated questions
